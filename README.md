# MyLittleFarmSimulator
 Небольшой концепт симулятора фермы.

## Задание
 Необходимо разработать демо-версию (концепт) игры "Симулятор фермы".
 Все выполнено самостоятельно. Не использовано сторонних пакетов или ассетов.
 
## Управление
 WASD
 LShift - Бег
